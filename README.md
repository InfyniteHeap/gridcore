# gridcore

The core of GCL that implements almost all of launcher functions, e.g. Microsoft OAuth2, Minecraft downloading and game mods managing.

## Usage

Add this crate in your `Cargo.toml` file under `[dependencies]` table:

```toml
[dependencies]
gridcore = "0.1.0"
```

Most of use methods can be found in `tests` folder.
