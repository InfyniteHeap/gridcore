# gridcore
The core of GCL that implements almost all of launcher functions.

# Usage
Please add this crate in your `Cargo.toml` file under `[dependencies]` item:

```toml
[dependencies]
gridcore = "0.1.0"
```

Most of use methods can be found in `tests` folder.
